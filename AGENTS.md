# AGENTS.md

Project: Scouts Maison de La Paix website

## Goal
Build a modern, warm, and human website for a scouting organization centered on peace, community, and youth development.

## Stack
- Next.js
- React
- TypeScript
- Tailwind CSS

## Design principles
- Clean, spacious, modern
- Warm, not cold
- Not a generic NGO site
- Not a playful kids-only site

## Branding
- Main association logo is primary identity
- Moroccan Scouting League is secondary (credibility only)
- Do NOT use the full banner in the hero

## Colors
- Forest Green: #264D3B
- Ivory: #F7F3EC
- Sand: #E9DFCF
- Terracotta: #B86A4A
- Charcoal: #2A2A2A

## Typography
- Fraunces for headings
- Manrope for body

## Coding style
- Simple and readable
- Use reusable components
- Comment code for beginner clarity
